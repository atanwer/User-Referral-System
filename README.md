# User Referral System

## Prerequisites

- Node.js: 20.17.0

## Database

- Mongodb

## Setup

```bash
git clone https://github.com/atanwer/User-Referral-System.git
cd User-Referral-System
npm install
```

### Create .env in root directory and add env variable by following env.example file

## Running Scripts

### Production

```bash
npm start
```

### Developement

```bash
npm run dev
```
